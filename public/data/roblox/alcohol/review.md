# What does Alcohol Software Offer?
Alcohol Software offers a free dumper, also called PasteDump.
Pastedump is an external tool that scans a running Roblox process to find the specific memory addresses of the game's core functions and data. It works by searching for unique "fingerprints" of code called patterns and following text strings to see where they are used in the engine. Once found, it calculates these locations as offsets so developers can use them to build software that interacts with the game.

Our last test indicates that **PasteDump v1.26** can successfully extract offsets from the Roblox client, allowing developers to interface with the game's internal structure for purposes such as creating custom tools or modified clients.
